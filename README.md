# prompt.txt
Design and implement a menu-driven 2D Graphics Editor in C using a 2D character array as the drawing canvas.
Requirements:
The canvas should consist of 25 rows and 50 columns.
Initially, all cells of the canvas must be filled with the character '_' (underscore).
Graphical objects should be drawn using the character '*' (asterisk).
Implement functions to:
Draw a Rectangle
Draw a Horizontal/Vertical Line
Draw a Triangle
Draw a Circle
Provide a function to display the canvas on the screen.
Allow the user to delete a rectangle by replacing its area with underscores.
Create a menu-driven interface that allows users to:
Add graphical objects
Delete objects
View the current canvas
Exit the program
Use appropriate functions, loops, and arrays to organize the program.
